# N — The backlog as a cascade: the build-out prior, typed

**Status: NOTE 2026-07-12 (claude-2, from Joe: "what if we characterised the
backlog as a cascade? … then we'd have an overall *prior* of how we think
things might develop"). Companion to the forward-model sequencing plan

## The move

A build-out sequencing plan and a pattern-cascade are the same shape:

| cascade (tactical, batch loop) | backlog-cascade (strategic) |
|---|---|
| box = a pattern applied | box = a mission advanced |
| wire = grounded composition | wire = dependency/enablement edge (census cross-refs) |
| psi (the want) | the strategic want: "a stack that closes its loops and feeds its operator" |
| satiety (:full/:partial) | how much of its wave-role the mission's current phase already delivers |
| absorption (same-move) | same-move missions (goals-and-holes ⇄ operational-vocabulary — the census found real honest-overlap pairs) |
| policy-hole = no pattern grounds this | **hole = no MISSION grounds this** — a mission that should exist and doesn't |
| terminals | the missions whose completion discharges the strategic want directly |
| fold discipline: wires earned, holes first-class | plan discipline: waves justified by edges, gaps named |

Three things fall out immediately:

1. **The prior becomes explicit and typed.** "How we think things might
   develop" stops being prose waves and becomes a cascade object — one the
   fold conventions (satiety honesty, absorption species, named holes)
   already discipline. The hand-authored sequencing v1 is revealed as
   **backlog-cascade v0: one sample from the prior, authored by the
   operator-reviewer pair.**
2. **The strategic level acquires temporal depth the paper already asked
   for.** main-2026.tex names the remaining hierarchy step exactly:
   *"surfacing the whole ranked plan, not just its head, so the
   recommendation reads as — work on this next; it is step one of this
   trajectory, because the whole trajectory minimises G."* A
   backlog-cascade IS that surfaced trajectory; G(π) at the strategic grain
   is a path integral over it (R13/R15).
3. **The proposer machinery applies at the strategic grain.** A
   distribution over backlog-cascades is exactly what the slush learns over
   pattern-cascades. The same A/B discipline (blind, preregistered,
   adjudicated) could one day compare backlog-cascade proposers; the
   roadmap.edn generative pass is already the mint lane at this grain
   (candidate MISSIONS entering at the uniform prior); and mission-level
   policy-holes are the mint lane's demand signal.

## Interaction with the held-out rediscovery test (discipline, stated now)

Holes in a backlog-cascade authored by the reviewer must be **derived from
in-plan evidence** (a ladder rung no mission grounds; a verdict that
demands an unminted mission), never from seed knowledge — authoring a hole
that paraphrases a sealed seed would be steering. The honest form of
rediscovery under this design: a MECHANISM proposing a backlog-cascade
whose hole (or box) is seed-congruent. That is a sharper test than
top-K ranking, because a cascade also says *where in the trajectory* the
seed belongs.
